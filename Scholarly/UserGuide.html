User Guide

After login / register, you will get on the home page, here you can choose what category of games you want to learn about.



Then, in each category you will see the games and for each game you will have to select the difficulty of the rules to learn (easy, medium, hard). Afterwards, the rules appear on the screen and, after reading the desired one, you can press the tick button to mark it as read so it counts towards your progress



If you want, you can change your password in the "My account" tab.



You can take tests on the "Test your knowledge" tab.



The home page for the user.



User can choose a category.



User then can choose the game and difficulty he wants.



The user will get questions.



The user will know how many wrong and correct answer he has and can see the correct ones.



After the test is ended, the user can see his results.



Admin Guide

First of all, you should login with your user and password (which was given to you by the developer). The application knows you are an admin if you login with an @admin domain, and gives you a JWT.



The admin page has 4 main panels, where you can view which game each user is learning about, change users' password, view a top of the users per game category.
